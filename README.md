# @hotmanxp/plugins-market

Monorepo containing multiple Claude plugins, distributed via `git-subdir` sparse clone.

## Installation

```bash
# Add this marketplace
/plugin add hotmanxp/plugins-market

# Install plugins
/plugin install superpowers@hotmanxp/plugins-market
/plugin install example-plugin@hotmanxp/plugins-market
```

## Plugins

| Plugin | Description |
|--------|-------------|
| superpowers | Core skills library: TDD, debugging, collaboration patterns, and proven techniques |
| example-plugin | Example plugin demonstrating git-subdir structure |
| typescript-lsp | TypeScript language server (typescript-language-server) for Claude Code |
| rust-analyzer-lsp | Rust Analyzer LSP for code intelligence for Rust projects |
| pyright-lsp | Pyright language server for Python code intelligence |
| jdtls-lsp | Eclipse JDT Language Server for Java code intelligence |
| kotlin-lsp | Kotlin Language Server for Kotlin code intelligence |
| gopls-lsp | Go language server (gopls) for Go code intelligence |
| swift-lsp | Swift language server (SourceKit-LSP) for Swift code intelligence |
| feature-dev | 7-phase structured feature development workflow with specialized agents |
